---
description: Mở Giao diện Graph Web bằng Cytoscape (VibeCode Visuals).
---

1. Sử dụng công cụ `run_command` để gọi kịch bản sinh giao diện tĩnh và mở trình duyệt thông qua lệnh sau:
// turbo
```bash
.\vibecode.bat ui
```
2. Lệnh này sẽ tự động sinh tệp tin `vibecode_graph.html` và chèn nội dung Cytoscape JSON. Môi trường Window mặc định sẽ mở luôn Chrome/Edge, hãy viết một dòng nhỏ cho người dùng xác nhận rằng giao diện đã được bật lên trình duyệt.
